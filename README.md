<h1>Description</h1>
<p>Pinky Jump is a small 2D side-scrolling platform game in which you control a
little pink character, with whom you must reach the goal located in the castle at the end of
the level by jumping across different platforms, facing enemies and collecting as many
coins as you can.</p>

<h1>Controls</h1>
<p>AD -> Left and Right directions.</p>
<p>Space Bar -> Jump.</p>

<h1>Main technical challenges</h1>
<p>Main technical challenges within the game: correct collision with the environment, making the game fluid, gravity control, and element spawning. </p>
<p>Outside of the game: working with SFML and TMX for the first time and trying to have a well-structured project.</p>
